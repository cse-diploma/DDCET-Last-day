Definition & Characteristics
What is a Computer?
A Computer is an electronic device that: • Accepts INPUT (data) • PROCESSES it (according to instructions/program) • Gives OUTPUT (result/information) • STORES data and instructions Full form: Commonly Operated Machine Particularly Used for Technical, Educational and Research purposes

Characteristics: Speed (nanoseconds), Accuracy (100%), Diligence, Versatility, Storage, Automation

Memory Types
Primary Memory (Internal)
RAM — Random Access Memory Volatile (data lost on power off) Readable & Writable Types: SRAM, DRAM, SDRAM ROM — Read Only Memory Non-volatile (permanent) Read only Types: PROM, EPROM, EEPROM Cache — Fastest memory, between CPU & RAM

Objective MCQ Points — Computer Basics
1. The 'brain' of the computer is called?
CPU (Central Processing Unit)
2. Which memory is volatile?
RAM — data is lost when power is turned off
3. 1 Kilobyte = ?
1024 Bytes (NOT 1000 Bytes)
4. Which is NOT an input device?
Monitor / Printer (these are output devices)
5. First generation computers used?
Vacuum Tubes (1940–1956)
6. Full form of BIOS?
Basic Input Output System
7. Binary number system has base?
Base 2 (uses only 0 and 1)
8. Decimal 10 in binary = ?
1010 (8+2=10 → 1×8 + 0×4 + 1×2 + 0×1)
9. Which part of CPU performs calculations?
ALU — Arithmetic Logic Unit
10. ROM is?
Non-volatile, Read Only Memory — data is permanent
11. DVD storage capacity?
4.7 GB (single-layer)
12. Which is fastest memory?
Cache Memory (fastest), then RAM, then Secondary storage
13. GUI stands for?
Graphical User Interface (e.g., Windows desktop)
14. Which printer is impact type?
Dot Matrix Printer (strikes ribbon against paper)
15. Compiler vs Interpreter?
Compiler: translates whole program at once. Interpreter: translates line by line
★ Memory Trick — Storage Units
Remember: B N B K M G T P E → Bit, Nibble, Byte, Kilo, Mega, Giga, Tera, Peta, Exa. Each step × 1024.
Trick — Binary Conversion Fast
Remember powers of 2: 1,2,4,8,16,32,64,128,256,512,1024. For any number, subtract the largest power of 2 that fits, place 1, continue. Example: 25 = 16+8+1 = 11001
Fast MCQ Rule
RAM = Volatile = Temporary. ROM = Non-volatile = Permanent. If question says "data lost on power off" → RAM. "Stores BIOS" → ROM.
Common Trap
1 KB = 1024 Bytes (NOT 1000). This is the most common trap in MCQs. Similarly 1 MB = 1024 KB, not 1000 KB.
Important — Generations
Exam love testing: "Which generation used transistors?" → 2nd Gen. "Which uses microprocessor?" → 4th Gen. "AI generation?" → 5th Gen.
Input vs Output Shortcut
Input = gives data TO computer (keyboard, mouse, scanner). Output = computer gives data OUT (monitor, printer, speaker). Touch screen and modem = BOTH.

Network Coverage Order
Remember smallest to largest: PAN → LAN → MAN → WAN. Pocket → Local → Metro → World.
Email Protocol Trick
SMTP = Send Mail To People (sending). POP3 = downloads to your device. IMAP = stays on server (accessible everywhere).
Browser ≠ Search Engine
Chrome/Firefox/Edge = Browsers (software to navigate). Google/Bing/Yahoo = Search Engines (finds content). This MCQ trap appears very frequently!
URL Protocol Trick
HTTP = normal. HTTPS = secure (has S for Secure). Always use HTTPS for banking/passwords. URL starts with protocol (http/ftp/mailto).
Virus Types to Remember
Virus=needs host. Worm=self-replicates via network. Trojan=disguised. Spyware=monitors. Ransomware=locks files for ransom. Adware=shows unwanted ads.

HTML Document Order
Remember the nesting order: <html> → <head> → <title> and <body>. Body has all visible content. Head has metadata (not visible).
Self-Closing Tags Memory Trick
Self-closing (void) tags: br hr img input meta link. Think "BHIIM L" — these don't need closing tags.
Table Tags Order
Table structure: <table> → <tr> (row) → <th> (header) or <td> (data). Always table first, then row, then cell.
Common Exam Trap
<b> vs <strong>: both look bold but <strong> has semantic importance for SEO/accessibility. Similarly <i> vs <em>. Exams love this distinction.
colspan vs rowspan
colspan = merge COLUMNS (horizontal). rowspan = merge ROWS (vertical). "col = across, row = down".
Form Method Trick
GET = Get from server (search). Shows data in URL. POST = Post to server (login, register). Hides data. Use POST for sensitive info!

COUNT vs COUNTA
COUNT = only numbers. COUNTA = all non-empty (text + numbers). COUNTBLANK = empty cells. Remember: COUNT is strict about numbers!
IF Formula Structure
=IF(condition, value_if_TRUE, value_if_FALSE). Think: "IF this is true, do this, ELSE do that." Example: =IF(A1>=50,"Pass","Fail")
VLOOKUP Fast Memory
VLOOKUP = Vertical (looks down a column). Arguments: (What to find, Where to look, Which column to return, Exact or Approximate). Last argument: 0=Exact, 1=Approximate.
Absolute Reference Trap
Forget $ signs and your formula shifts when copied. For tax rates, constants, fixed values — always use $A$1. Press F4 to add $ signs automatically!
Chart Types to Remember
Bar/Column = compare values. Line = trends over time. Pie = parts of whole. Scatter = relationship between two variables. Area = cumulative values over time.
Error Types Quick Reference
#DIV/0! = divide by zero. #VALUE! = wrong data type. #REF! = invalid cell reference. #NAME? = wrong function name. #N/A = value not found (VLOOKUP). #NULL! = wrong range operator.

Animations vs Transitions
Animations
Applied to: OBJECTS on a slide (text boxes, images, shapes) Types: Entrance — objects appear (Fade, Fly In) Emphasis — objects highlight (Spin, Grow) Exit — objects disappear (Fade Out) Motion Path — objects move along a path Tab: Animations → Add Animation Animation Pane: Ctrl+Shift+A (or View Pane) Trigger: On Click / With Previous / After Previous
Transitions
Applied to: Slides (how one slide changes to next) Examples: Subtle: Cut, Fade, Push, Wipe, Split Exciting: Dissolve, Zoom, Cube, Flip Dynamic: Pan, Ferris Wheel, Gallery Options: Duration — speed of transition Sound — add sound effect Advance Slide — On Mouse Click or After X seconds Apply to All — apply same to all slides
Slide Layouts (Home → Layout)
Title Slide — Large title + subtitle (for first slide) Title and Content — Title + content area (most common) Two Content — Title + 2 side-by-side content areas Comparison — For comparing two items Title Only — Just the title placeholder Blank — No placeholders Content with Caption — For images with captions Picture with Caption — Full-picture + caption
Slide Size Options
Standard (4:3) — Older projectors (10×7.5 inches) Widescreen (16:9) — Modern screens (default since 2013) Custom Slide Size — Set custom dimensions Location: Design Tab → Slide Size

Animation vs Transition — Never Confuse
Animation = objects on a slide move/appear. Transition = slides change. Think: "Objects Animate, Slides Transition."
F5 vs Shift+F5
F5 = Start from slide 1 (beginning). Shift+F5 = Start from current slide. If you're on slide 3, Shift+F5 starts from slide 3.
Slideshow Keyboard Controls
During presentation: N or → = Next, P or ← = Previous, B = Black screen, W = White screen, Esc = End show. These are frequently tested!
Slide Layout vs Slide Master
Layout = arrangement of placeholders for ONE slide. Slide Master = design/format applied to ALL slides. Master is global, Layout is individual.
Animation Trigger Types
On Click = plays when you click. With Previous = plays simultaneously with previous animation. After Previous = plays after previous finishes automatically.
File Extensions Memory
Word = .docx, Excel = .xlsx, PowerPoint = .pptx. All modern MS Office files end in 'x' (for XML). Older versions: .doc, .xls, .ppt.
