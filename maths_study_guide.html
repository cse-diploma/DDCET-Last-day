Determinant Formulas
1×1 & 2×2 Determinant
|a| = a |a b| |c d| = ad - bc

Rule: Multiply main diagonal, subtract anti-diagonal product

3×3 Determinant (Expansion along Row 1)
|a₁ b₁ c₁| |a₂ b₂ c₂| = a₁(b₂c₃ - b₃c₂) - b₁(a₂c₃ - a₃c₂) + c₁(a₂b₃ - a₃b₂) |a₃ b₃ c₃|

Signs: + − + for first row. Cofactor expansion method.

Matrix Operations
Addition / Subtraction
(A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ Condition: Same order (m×n)

Commutative: A+B = B+A | Associative: (A+B)+C = A+(B+C)

Scalar Multiplication
(kA)ᵢⱼ = k · aᵢⱼ k(A+B) = kA + kB (k+l)A = kA + lA
Matrix Multiplication
(AB)ᵢⱼ = Σₖ aᵢₖ · bₖⱼ Condition: Columns of A = Rows of B If A is m×n and B is n×p → AB is m×p

NOT commutative (AB ≠ BA in general). Associative: (AB)C = A(BC)

Adjoint & Inverse of 2×2 Matrix
For A = |a b; c d|
adj(A) = | d -b| |-c a| A⁻¹ = adj(A) / |A| = (1/(ad-bc)) × | d -b| |-c a| Condition: |A| ≠ 0 (non-singular) Verify: A × A⁻¹ = I

Formula: Swap main diagonal, negate off-diagonal, divide by determinant

Simultaneous Linear Equations (Cramer's Rule)
Two Variables: a₁x + b₁y = c₁ | a₂x + b₂y = c₂
D = |a₁ b₁| = a₁b₂ - a₂b₁ |a₂ b₂| Dₓ = |c₁ b₁| = c₁b₂ - c₂b₁ |c₂ b₂| Dᵧ = |a₁ c₁| = a₁c₂ - a₂c₁ |a₂ c₂| x = Dₓ/D , y = Dᵧ/D (when D ≠ 0)
Matrix Method: AX = B → X = A⁻¹B
|a₁ b₁| |x| |c₁| |a₂ b₂| |y| = |c₂| [x] = A⁻¹ [c₁] [y] [c₂]

Tip
For 2×2 inverse: just remember "SWAP the diagonal, NEGATE the off-diagonal, DIVIDE by det". No need to compute cofactors separately.
Trick — Determinant of 3×3
Use Sarrus' Rule: Write the first two columns again to the right of the matrix. Multiply left-to-right diagonals (+), subtract right-to-left diagonals (−). Very fast for numerical problems.
Fast Method — Cramer's Rule
Replace x-column with constants for Dₓ, replace y-column with constants for Dᵧ. The pattern is: constant column replaces the variable's column.
Tip
Check: if any two rows (or columns) are identical in a determinant → value = 0. If a row is all zeros → determinant = 0.
Common Mistake
AB ≠ BA in general. Do NOT apply commutativity to matrix multiplication. Always check order.
Trick — Verify Inverse
Always verify: A × A⁻¹ = I. If not, recheck your determinant or adj calculation.
Fast Check — Singular?
For 2×2 |a b; c d|: it's singular if ad = bc (i.e., rows/columns are proportional).

Mnemonic — CAST Rule
In which quadrant are functions positive? All Students Take Calculus (Q1: All, Q2: Sin, Q3: Tan, Q4: Cos). Quadrant I→IV going counterclockwise.
Trick — Values by Hand
For sin(0°,30°,45°,60°,90°): numerators are √0,√1,√2,√3,√4 divided by 2. That gives 0, ½, 1/√2, √3/2, 1. For cos, reverse the order!
Fast — Allied Angle Rule
For multiples of 90°: function changes (sin↔cos, tan↔cot). For multiples of 180°: function stays same. Sign depends on quadrant of original angle.

Basics
Vector Notation
a⃗ = aₓî + aᵧĵ + a_zk̂ where î,ĵ,k̂ are unit vectors along X,Y,Z
Magnitude (Length)
|a⃗| = √(aₓ² + aᵧ² + a_z²) In 2D: |a⃗| = √(aₓ² + aᵧ²)

Vector Operations
Addition & Subtraction
a⃗ + b⃗ = (aₓ+bₓ)î + (aᵧ+bᵧ)ĵ + (a_z+b_z)k̂ a⃗ − b⃗ = (aₓ−bₓ)î + (aᵧ−bᵧ)ĵ + (a_z−b_z)k̂ Triangle Law: a⃗ + b⃗ = c⃗ Parallelogram Law: |R⃗|² = P²+Q²+2PQcosθ
Dot Product (Scalar Product)
a⃗ · b⃗
a⃗ · b⃗ = |a⃗||b⃗| cosθ = aₓbₓ + aᵧbᵧ + a_zb_z (component form) cosθ = (a⃗ · b⃗) / (|a⃗| · |b⃗|) Properties: • Commutative: a⃗ · b⃗ = b⃗ · a⃗ • Result is a SCALAR • a⃗ · a⃗ = |a⃗|² = aₓ² + aᵧ² + a_z² • If perpendicular: a⃗ · b⃗ = 0 (cosθ = 0) • î·î = ĵ·ĵ = k̂·k̂ = 1 • î·ĵ = ĵ·k̂ = k̂·î = 0
Cross Product (Vector Product)
a⃗ × b⃗
a⃗ × b⃗ = |a⃗||b⃗| sinθ · n̂ Component form (determinant): a⃗ × b⃗ = | î ĵ k̂ | | aₓ aᵧ a_z| | bₓ bᵧ b_z| = î(aᵧb_z − a_zbᵧ) − ĵ(aₓb_z − a_zbₓ) + k̂(aₓbᵧ − aᵧbₓ) |a⃗ × b⃗| = |a⃗||b⃗| sinθ (magnitude = area of parallelogram) Properties: • NOT commutative: a⃗ × b⃗ = −(b⃗ × a⃗) • Result is a VECTOR (perpendicular to both) • If parallel: a⃗ × b⃗ = 0⃗ • î×ĵ = k̂, ĵ×k̂ = î, k̂×î = ĵ • ĵ×î = −k̂, k̂×ĵ = −î, î×k̂ = −ĵ • î×î = ĵ×ĵ = k̂×k̂ = 0⃗
Angle Between Vectors
cosθ = (a⃗ · b⃗)/(|a⃗| · |b⃗|) θ = 0° → parallel (same direction) θ = 90° → perpendicular θ = 180° → anti-parallel (opposite direction) Condition for perpendicular: a⃗ · b⃗ = 0 Condition for parallel: a⃗ × b⃗ = 0⃗

Mnemonic — Cross Product Signs
Cyclic order: î → ĵ → k̂ → î. Forward = positive (î×ĵ=k̂). Backward = negative (ĵ×î=−k̂).
Trick — Cross Product 3×3
Always expand the 3×3 determinant along the first row (î, ĵ, k̂). Remember sign pattern: + − +.
Fast Method — Angle between Vectors
1) Compute dot product, 2) Compute both magnitudes, 3) Divide. That gives cos θ. Take inverse cosine for the angle.
Key Difference
Dot product → scalar (one number). Cross product → vector. If question asks for a number result, it's likely dot product. If it asks for a perpendicular vector, it's cross product.

Coordinate Geometry
Lines, slopes, forms of equations, parallel/perpendicular, circles
4 Marks
Slope of a Line
m = tanθ (θ = inclination angle with positive x-axis) m = (y₂ − y₁)/(x₂ − x₁) (two-point formula)
Equation of a Line — All Forms
Two-Point Form
(y − y₁)/(y₂ − y₁) = (x − x₁)/(x₂ − x₁)
Slope-Point Form
y − y₁ = m(x − x₁) where m = slope, (x₁,y₁) = given point
Slope-Intercept Form
y = mx + c (c = y-intercept, m = slope)
Intercept Form
x/a + y/b = 1 (a = x-intercept, b = y-intercept)
General Form
ax + by + c = 0 Slope m = −a/b x-intercept = −c/a, y-intercept = −c/b
Parallel & Perpendicular Lines
Parallel Lines
Condition: m₁ = m₂ (equal slopes) Line through (x₁,y₁) parallel to ax+by+c=0: ax + by = ax₁ + by₁
Perpendicular Lines
Condition: m₁ × m₂ = −1 Line through (x₁,y₁) perpendicular to ax+by+c=0: bx − ay = bx₁ − ay₁
Angle Between Two Lines
tanθ = |(m₁ − m₂)/(1 + m₁m₂)| If m₁m₂ = −1 → lines are perpendicular (θ = 90°) If m₁ = m₂ → lines are parallel (θ = 0°)
Distance Formulas
Distance between two points: d = √((x₂−x₁)² + (y₂−y₁)²) Distance from point (x₁,y₁) to line ax+by+c=0: d = |ax₁ + by₁ + c| / √(a² + b²) Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
Circle Equations
Standard Form
(x − h)² + (y − k)² = r² Centre = (h, k), Radius = r If centre at origin (0,0): x² + y² = r²
General Form → Centre & Radius
General form: x² + y² + 2gx + 2fy + c = 0 Centre = (−g, −f) Radius = √(g² + f² − c) Condition: g² + f² − c > 0 (real circle)

Remember: centre coordinates are negatives of g and f (the half-coefficients)

Tip — General Circle Form
To convert to standard form: complete the square for x and y. Group x terms, add (b/2)² to both sides, repeat for y.
Trick — Perpendicular Slope
Perpendicular slope = negative reciprocal. If m = 2/3, perpendicular slope = −3/2. Just flip and negate!
Fast — Line Through Point, Parallel to ax+by+c=0
Just replace c with −(ax₁+by₁). New line: ax + by − (ax₁+by₁) = 0 or ax+by = ax₁+by₁.
Common Mistake
In general circle x²+y²+2gx+2fy+c=0, centre is (−g,−f) NOT (g,f). The negative sign is crucial.
Trick — Two Point to Slope-Point
First find slope m = (y₂−y₁)/(x₂−x₁), then substitute ONE point into y−y₁=m(x−x₁). Simpler than remembering the two-point formula directly.

Rules of Differentiation
Sum / Difference Rule
d/dx [f±g] = f' ± g'
Scalar Multiple Rule
d/dx [kf(x)] = k·f'(x)
Product Rule
d/dx [u·v] = u·v' + v·u' (First × Derivative of Second) + (Second × Derivative of First)
Quotient Rule
d/dx [u/v] = (v·u' − u·v') / v² Denominator²
Chain Rule
dy/dx = (dy/du) × (du/dx) d/dx [f(g(x))] = f'(g(x)) · g'(x) Example: d/dx [sin(x²)] = cos(x²) · 2x
Implicit Differentiation
If f(x, y) = 0, differentiate both sides w.r.t. x. Treat y as function of x. Use chain rule for y terms. d/dx[y²] = 2y · (dy/dx) d/dx[y³] = 3y² · (dy/dx) Then solve for dy/dx.
Parametric Differentiation
If x = f(t) and y = g(t): dy/dx = (dy/dt)/(dx/dt) = g'(t)/f'(t)
Logarithmic Differentiation
For y = [f(x)]^g(x): Take log: ln y = g(x) · ln f(x) Differentiate both sides: (1/y)(dy/dx) = g'(x)·ln f(x) + g(x)·f'(x)/f(x) Multiply both sides by y.
Successive Differentiation
First derivative: y' = dy/dx = f'(x) Second derivative: y'' = d²y/dx² = f''(x) = d/dx[f'(x)]
Applications — Velocity & Acceleration
If position s = f(t)
Velocity v = ds/dt = f'(t) Acceleration a = dv/dt = d²s/dt² = f''(t) At rest: v = 0 (solve for t) Uniform: a = 0 (constant velocity)

Logarithm as function, laws of logarithm
4 Marks
Definition
log_b(x) = y ↔ bʸ = x (b is base, b > 0, b ≠ 1, x > 0) Natural log: ln x = log_e x (base = e ≈ 2.71828) Common log: log x = log₁₀ x (base = 10)
Laws of Logarithm
All Logarithm Rules
1. Product Rule: log(mn) = log m + log n 2. Quotient Rule: log(m/n) = log m − log n 3. Power Rule: log(mⁿ) = n·log m 4. Base Change: log_b(x) = log_a(x)/log_a(b) 5. log_b(b) = 1 (log of base to itself = 1) 6. log_b(1) = 0 (log of 1 = 0, any base) 7. log_b(bⁿ) = n 8. b^(log_b x) = x (exponential-log inverse) 9. log_a(x) = 1/log_x(a) (reciprocal property) 10. log₁₀ 2 ≈ 0.3010, log₁₀ 3 ≈ 0.4771 Important: log_b(0) = undefined, log of negative = undefined

Golden Rule
Three core rules: log of product = sum, log of quotient = difference, log of power = multiply. All simplification problems use only these three.
Trick — Base Change
To convert log₂(8) to common log: = log(8)/log(2) = 3log2/log2 = 3. Very useful to simplify without a calculator.
Fast Values
Memorize: log 2 = 0.3010, log 3 = 0.4771. Then: log 4 = 2×0.3010 = 0.6020, log 6 = log2+log3 = 0.7781, log 5 = log(10/2) = 1−0.3010 = 0.6990.
Warning
log(m+n) ≠ log m + log n. Only log(m×n) = log m + log n. This is the most common misconception in log problems.

Mean, Median, Mode for ungrouped data
2 Marks
Measures of Central Tendency
Mean (Arithmetic Mean)
x̄ = (x₁ + x₂ + ... + xₙ)/n = Σxᵢ/n (Sum of all values divided by count)
Median
Step 1: Arrange data in ascending order Step 2: If n is ODD: Median = ((n+1)/2)th term If n is EVEN: Median = [(n/2)th + (n/2+1)th] / 2
Mode
Mode = value that occurs MOST frequently • Unimodal: one mode • Bimodal: two modes • Multimodal: more than two modes • No mode: all values occur equally
Empirical Relation
Mode = 3·Median − 2·Mean (For moderately skewed data)
Worked Example
Data: 4, 7, 3, 9, 7, 2, 7, 1 Sorted: 1, 2, 3, 4, 7, 7, 7, 9 (n = 8, even) Mean = (1+2+3+4+7+7+7+9)/8 = 40/8 = 5 Median = (4th + 5th)/2 = (4+7)/2 = 5.5 Mode = 7 (appears 3 times)

Tip — Step by Step
Mean: add all, divide by n. Median: sort first, then find middle. Mode: find most repeated. Remember this order!
Trick — Quick Median
For n=5 terms: median is 3rd term. For n=7: 4th term. For n=9: 5th term. Formula: (n+1)/2 for odd n gives term position directly.
Fast — Mean
If all values are close, pick a working mean (approximate), find deviations, average deviations, add back. Faster arithmetic for messy numbers.
Always Sort First!
The most common mistake: forgetting to arrange data in ascending order before finding the median. Median on unsorted data = wrong answer.
